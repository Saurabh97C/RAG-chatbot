⚡ RAG Chatbot (NEC + Wattmonk)

📌 Overview
This project is a Retrieval-Augmented Generation (RAG) chatbot that answers user queries using:
- NEC (National Electrical Code) data
- Wattmonk company information

The chatbot retrieves relevant context from documents and generates answers using an LLM.

🚀 Features
- Multi-document question answering
- Semantic search using FAISS
- Context-aware response generation
- Streamlit-based interactive UI
- Lightweight models for efficient performance

🛠 Tech Stack
- Python
- LangChain
- FAISS (Vector Database)
- HuggingFace Transformers (Flan-T5)
- Sentence Transformers (MiniLM)
- Streamlit

▶️ How to Run: https://rag-chatbot-saurabh.streamlit.app/
